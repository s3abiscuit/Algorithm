# about sort

## interface

- Comparable(compareTo)

## sort method

1. selection: 每次选择一个最小的
2. insertion: 每次选择一个插入到之前有序数列
3. shell: 多次插入排序, 多一轮循环
4. merge: sort, sort, merge
5. quick: partition, sort, sort
6. heap: consturct heap, exchange
